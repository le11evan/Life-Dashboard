# /next Command

Implement the next smallest shippable slice end-to-end.

## Instructions

1. Read CLAUDE.md for current milestone and progress
2. Identify the smallest complete feature that can ship
3. Implement it fully:
   - Database schema if needed
   - Server actions/API
   - UI components
   - Animations
4. Test locally
5. Run: `npm run lint && npm run typecheck && npm run build`
6. If all pass, commit with clear message
7. Update CLAUDE.md with progress
